## Fabric invariants

1. **Exchange stays fast** — no new synchronous dependencies on the token exchange pipeline.
2. **Revocation stays fast** — kill-switch propagation latency is not negotiable.

Async surfaces ([dashboard](/1.0/docs/integrators/dashboard/), [graph](/1.0/docs/integrators/starfly-graph/), [UTC](/1.0/docs/integrators/utc/)) observe the fabric — they never block exchange or revocation.

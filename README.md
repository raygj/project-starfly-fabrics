# project-starfly-fabrics

Public website for [Starfly](https://starfly.dev) — deployed to Cloudflare Workers (static assets).

Source code and docs live in the monorepo: [raygj/project-starfly](https://github.com/raygj/project-starfly).

## Cloudflare deploy

| Field | Value |
|---|---|
| Repository | `raygj/project-starfly-fabrics` |
| Project name | `project-starfly-fabrics` |
| Build command | *(empty)* |
| Deploy command | `npm install && npx wrangler deploy` |

Custom domain: add `starfly.dev` under **Workers & Pages → project-starfly-fabrics → Domains**.
